# MLP HUNsub

[My Little Pony Friendship Is Magic](https://en.wikipedia.org/wiki/My_Little_Pony:_Friendship_Is_Magic) Magyar feliratok

A repó tartalmazza az összes feliratot a sorozathoz UTF-8 kódolt SRT formátumban amely a [Hunbrony](hunbrony.hu) égisze alatt publikálásra került valaha.

## Tartalom

Évadonként külön mappában találod az epizódokhoz tartozó feliratokat. A sima `season_X` mappák az eredeti, Hunbrony csapat munkájaként publikált feliratok. De történészeink felfedeztek egy alternatív feliratot is az első évadból, amelyet Alterego-X készített, ezt a `season_1_alt` mappában találod.
